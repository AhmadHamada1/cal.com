Google Sheets integration for Cal.com allows you to automatically send booking data to Google Sheets. When enabled for an event type, new bookings will be added as rows to your selected Google Sheet.

## Features

- Automatically send booking data to Google Sheets
- Configure which Google Sheet to use per event type
- Choose which booking fields to include in your sheet
- Create new sheets or use existing ones
- Secure OAuth authentication with Google

## How it works

1. Install the Google Sheets app and connect your Google account
2. Enable the app for specific event types
3. Configure which Google Sheet to use
4. When new bookings are made, they'll automatically be added to your sheet
