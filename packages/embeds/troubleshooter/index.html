Cal.com Embed Troubleshooter

A diagnostic tool for troubleshooting Cal.com embeds on any website. This bookmarklet helps you quickly identify issues with Cal.com embed installations, configuration problems, and provides recommendations for fixes.

Install Bookmarklet

Drag this button to your bookmarks bar:

Cal.com Troubleshooter

How to use:

  1. Drag the button above to your browser's bookmarks bar
  2. Navigate to any page with Cal.com embeds
  3. Click the bookmarklet to open the troubleshooter
  4. Review the diagnostic results and recommendations

Features

✓ Detects if window.Cal is properly initialized
✓ Identifies all Cal.com embed elements on the page
✓ Monitors network requests to Cal.com
✓ Captures console errors related to embeds
✓ Checks for configuration issues
✓ Validates security policies (CSP, CORS)
✓ Provides actionable recommendations
✓ Real-time monitoring with refresh capability

Development

To test locally, you can copy the contents of bookmarklet-loader.js and paste it into your browser's console on any page with Cal.com embeds.
